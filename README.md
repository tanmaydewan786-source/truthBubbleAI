# Truth Bubble AI

This is a Vite + React app for the Truth Bubble AI landing page.

## Run locally

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Install dependencies:
   ```bash
   npm install
   ```
2. Build and publish the site:
   ```bash
   npm run deploy
   ```
3. In GitHub, open your repository settings and enable GitHub Pages.
4. Set the source to the `gh-pages` branch.

Your site will be available at:

```text
https://<your-username>.github.io/truthBubbleAI/
```
